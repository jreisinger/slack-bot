Slack bot
=========

Install
-------

Install node from https://nodejs.org/en/download/

Then:
```
npm install             # install deps from package.json
npm install -g nodemon  # optional
```

Store token in `config.json`:

```
{
    "token": "TOKEN"
}
```

Run
---

```
node index.js
```
