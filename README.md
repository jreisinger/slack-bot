Slack bot
=========

Installation
------------

Install node from https://nodejs.org/en/download/

Then:
```
npm install -g nodemon
npm install  # install deps from package.json
```

Store toke in `config.json`:

{
    "token": "<token>"
}


Running
-------

```
node index.js
```
